# Mini E-Commerce App (Expo + React Native)

A clean, modern, and animated shopping cart application built with **Expo**, **React Native**, **Zustand**, and **Fake Store API**.

Beautiful UI with smooth animations (Reanimated v3/v4), quantity controls, cart management, and responsive product grid.


## ✨ Features

- Browse products from Fake Store API (grid layout)
- Product detail screen with rating, description & "Add to Cart"
- Persistent shopping cart using Zustand
- Quantity increase/decrease + remove item
- Real-time cart total & item count badge
- Smooth animations (fade, spring, zoom) on cart interactions
- Empty state, loading spinner, error handling
- Modern card design with shadows, press feedback & neumorphism-inspired elements
- Currency formatting (Indian Rupee ₹)

## 🛠️ Tech Stack

- **Expo** (SDK 50+ / 51 / 54 / 55 compatible)
- **React Native** (latest stable)
- **TypeScript**
- **Zustand** – lightweight state management
- **React Navigation** (bottom tabs + stack)
- **React Native Reanimated** v3/v4 – buttery smooth animations
- **Expo Vector Icons** (Ionicons)
- **Numeral.js** – number formatting
- **Axios** – API calls
- **Fake Store API** – https://fakestoreapi.com
